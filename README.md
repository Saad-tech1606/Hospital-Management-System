## 🚀 About the Project

A **Hospital Management System** designed to streamline hospital operations including patient handling, doctor allocation, bed tracking, and blood bank management.

This project is built as a **collaborative full-stack system**, where:

* 🎨 Frontend is developed in this repository
* ⚙️ Backend is developed separately and integrated via APIs

---

## ✨ Key Features

* 🧑‍⚕️ Patient-to-Doctor Allocation
* 🛏️ Bed Availability & Management
* 🩸 Digital Blood Bank System
* 📊 OPD & Appointment Handling
* 🔄 Frontend–Backend API Integration
* ⚙️ Admin Panels for Management
